# Aviation English Learning Platform ✈️🇬🇧

Интерактивная Full-Stack веб-платформа, разработанная для специализированного изучения авиационного английского языка, подготовки к профессиональным тестам и расширения профильного словарного запаса авиаспециалистов.

Проект построен на современном и востребованном технологическом стеке с использованием модульной архитектуры на бэкенде и компонентного подхода на фронтенде.

## 🛠 Технологический стек

### Frontend
- **React** & **TypeScript** — разработка динамичного и типизированного интерфейса.
- **Vite** — быстрая сборка приложения.
- **TailwindCSS** (или ваш CSS-фреймворк) — адаптивная и современная верстка.

### Backend
- **NestJS** & **TypeScript** — модульный бэкенд-фреймворк с четкой архитектурой, обеспечивающий масштабируемость.
- **Prisma ORM** — типизированное взаимодействие с базой данных и управление миграциями.
- **PostgreSQL** — надежная реляционная СУБД.
- **JWT (JSON Web Tokens)** — безопасная аутентификация и авторизация пользователей.

### DevOps & Инструменты
- **Docker** & **Docker Compose** — контейнеризация бэкенда и базы данных для быстрой развертки в любом окружении.
- **Git / GitHub** — контроль версий.

---

## 🚀 Ключевой функционал

1. **Аутентификация и профили:** Регистрация, безопасный вход по JWT-токенам, разделение ролей (студент / администратор).
2. **Интерактивные модули обучения:** Карточки для изучения специализированной авиационной лексики, интерактивные упражнения.
3. **Тестирование:** Система тестов для проверки знаний с автоматическим подсчетом результатов.
4. **Админ-панель:** Возможность добавления и редактирования учебных материалов через бэкенд.
5. **База данных:** Прогресс пользователей, история прохождения тестов и словари хранятся в связанных таблицах PostgreSQL.

---

## 📦 Архитектура базы данных

Взаимодействие с данными организовано через **Prisma ORM**. Спроектированы связи:
- `One-to-Many` (Один-ко-многим) — например, связь пользователя с его результатами тестов.
- `Many-to-Many` (Многие-ко-многим) — связь между уроками и словарями.

Пароли пользователей перед сохранением в базу данных проходят обязательное хеширование.
